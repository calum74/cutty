# Print

Sample: [print.cpp](../samples/print.cpp)

Print any value, including ranges and values that lack stream and formatting support. Automatically prints the contents of ranges, or displays the type name and address for objects without iostream support.

```c++
std::vector<int> x;
cy::print("The values are", x);
```
